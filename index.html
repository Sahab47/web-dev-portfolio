Vanilla JavaScript

Vanilla JavaScript is the core language of the web—pure, unaltered JavaScript without any libraries or frameworks. It lets you manipulate the DOM, handle events, and build interactive websites with just your browser and your code.

React

React is a JavaScript library developed by Meta for building dynamic and responsive user interfaces using reusable components. It uses a virtual DOM to efficiently update only the parts of the UI that change, improving performance. With JSX syntax and one-way data binding, React makes it easier to manage complex application states and design scalable front-end architectures.

Node.Js

Node.js is an open-source, cross-platform JavaScript runtime environment that allows developers to run JavaScript code outside the browser, typically on servers. It uses an event-driven, non-blocking I/O model, making it highly efficient for building scalable and real-time applications like chat apps and APIs. Built on Chrome’s V8 engine, Node.js enables full-stack development using a single language—JavaScript—for both client and server sides.

mongobd

MongoDB is a popular open-source NoSQL database that stores data in flexible, JSON-like documents called BSON. It’s designed for scalability and high performance, making it ideal for handling large volumes of unstructured or semi-structured data. With features like indexing, replication, and sharding, MongoDB supports modern applications that require real-time access and distributed architecture.
